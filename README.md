
# 医療ドキュメントマネージャー

## 概要
このアプリは、自立支援医療・障害者手帳・障害年金の医療関連文書の期限管理を目的とした、  
React / Node.js / SQLite ベースの医療文書管理システムです。  
現場の業務効率化と更新漏れの防止を目的に、非エンジニアが自ら開発しています。

## 特徴
- 文書ごとの更新期限の自動管理
- 患者単位での記録整理
- モーダルUIでの情報登録・編集
- 保険変更や生保請求など多様なイベントに対応

## 技術スタック
- Frontend: React (Vite, TypeScript)
- Backend: Node.js (Express)
- DB: SQLite
- UI: Tailwind CSS

## 今後の開発予定
- 随時追加予定

## 背景
医療事務として働く中で、  
Excelで行っていた期限管理に限界を感じ、現場視点でのシステムをゼロから構築中。
